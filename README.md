# Prompts Node Libs 文档索引

本项目包含多个 prompt 文档，用于指导和规范化各类文档生成与开发流程。

## 文档列表

### 1. 生成文档

**功能**: 指导如何为项目生成规范化的文档结构

**适用场景**: 
- 需要为项目生成规范化的文档
- 需要创建项目概述文档 (summary.md)
- 需要创建 API 文档 (api.md)
- 需要确保文档格式统一规范

**核心内容**:
- 文档结构规范：包含 summary.md（项目概述）和 api.md（API 文档）
- 标题级别要求：仅使用 h3 (###) 及以下
- 表格格式规范：属性名、类型/参数、默认值/返回值、说明
- 内容准确性要求：基于实际代码分析

**使用方式**: 阅读 [原文档](./prompts/生成文档.md) 了解详情
